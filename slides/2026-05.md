# .NET MAUI News

## May 2026

### Date Range: 01. 05 -> 31. 05

---

# Executive summary

May was mostly about **MAUI 10 servicing and regression cleanup**.

- MAUI 10.0.70 was the main stable update
- Several 10.0.60 regressions needed attention
- MAUI 11 Preview 4 showed upcoming build-host requirements
- CommunityToolkit.Maui 14.1.1 fixed Popup, StatusBar, and SpeechToText issues
- Apple May platform updates mean build-host pinning still matters

---

# Top 3 must-discuss items

1. **MAUI 10.0.70 upgrade plan**  
   Good servicing value, but test before shipping.

2. **10.0.60 regression impact**  
   Check Shell, WebView, gradients, app icons, visual states, and CollectionView.

3. **Preview isolation**  
   Test MAUI 11 Preview 4 in a separate CI lane only.

# Framework / runtime updates

## Stable

- **MAUI 10.0.70** shipped on 21 May
- 135 commits focused on quality and platform fixes
- Key areas: Blazor Hybrid, Button/ImageButton, CollectionView, CarouselView, accessibility, Windows UI

## Preview

- **MAUI 11 Preview 4** shipped on 13 May
- Requires newer workload set and platform SDKs
- Useful for migration checks, not production

---

# Tooling / platform updates

- MAUI 11 Preview 4 points to:
  - Xcode 26.4
  - JDK 21.0.8
  - Android API 36
  - Windows App SDK 1.8
- Apple released May platform updates on 11 May
- No major May Visual Studio or Rider update was found that directly changed MAUI work

---

# Production risks and regressions

Watch these before upgrading:

- Android gradients can render wrong
- Shell navigation bar visibility can be wrong
- Android WebView can affect theme colors
- Empty SVG app icons can break builds
- Custom controls can stay stuck in Selected state
- Grouped CollectionView ScrollTo has platform issues
- Android WebView callback cleanup was fixed in servicing work

---

# Team action checklist

- Stage MAUI 10.0.70 before production rollout
- Test Shell navigation and back-button handling
- Test WebView behavior on Android
- Test grouped CollectionView scrolling on all target platforms
- Check app icon SVGs in CI
- Validate Toolkit Popup, StatusBar, and SpeechToText if used
- Pin SDK, workloads, Xcode, Java, and Android SDK
- Add separate MAUI 11 preview CI lane
