# Executive summary

The main theme is **safe upgrades and toolchain control**.

- MAUI 10.0.70 and 10.0.71 are the stable updates to test
- .NET 11 moves MAUI mobile toward CoreCLR by default
- Xcode, Android SDK, JDK, SDK, and workloads need pinning
- Main watch areas: CollectionView, Shell, WebView, Hot Reload, gradients

# Framework / runtime updates

- MAUI 10.0.70: broad stable quality update
- MAUI 10.0.71: targeted fixes for Shell, WebView, maps, tabs, keyboard, and Windows rendering
- .NET 11 Preview 4: CoreCLR becomes default for MAUI mobile
- .NET 11 Preview 5: more MAUI platform fixes and Android API 37 work

<!--
Message: stable track needs staging; preview track needs measurement.

---

# Tooling / platform updates

- MAUI 11 preview expects newer tools: Xcode 26.4, JDK 21.0.8, Android API 36
- Apple shipped Xcode 26.5, Xcode 26.6, Xcode 27 beta, TestFlight updates, and OS updates
- Android tooling moved; keep Release AAB/signing/device-install checks active
- VS and Rider had servicing updates, but no separate high-signal MAUI feature to act on

Main point: release lanes should not drift silently.

# Production risks and regressions

| Risk | Why it matters |
|---|---|
| CollectionView jank / GC | Can block feed, catalog, chat, and dashboard screens |
| Shell / modal / tab navigation | Real app flow risk |
| WebView / HybridWebView | Hybrid apps and Windows screens need checks |
| Hot Reload instability | Slows development and can confuse debugging |
| Gradients / icons / drawing | Visual regression risk |

# CommunityToolkit.Maui

Useful updates from the range:

- 14.1.1: Popup helper, StatusBar fixes, iOS SpeechToText fix
- 14.2.0: Popup v2 Android screen-reader accessibility
- MediaElement 10.0.0: stream media source support
- Camera 6.1.0: IsTorchOn and Windows camera permission fix

Action: upgrade in a branch and test only the features your app uses.

# Team action checklist

- Stage MAUI 10.0.71
- Add non-blocking .NET 11 / MAUI 11 CI
- Benchmark CoreCLR: startup, memory, app size, diagnostics
- Pin SDK, workloads, Xcode, JDK, Android SDK
- Test CollectionView, CarouselView, Shell, tabs, modals, WebView, maps, icons
- Validate Android AAB/signing/upload and iOS/Mac Catalyst signing/TestFlight
- Upgrade Toolkit packages in a branch if affected
